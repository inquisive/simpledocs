# SimpleDocs
### documentation builder

Built with ReactJS & KeystoneJS

[Demo / Documentation](https://simpledocs.inquisive.com)

### Install

```
npm install simpledocs

//or add to package.json
"dependencies": {
	"simpledocs": "~0.0.x"
}
```
